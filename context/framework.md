# Growth Marketing Framework

> The shared vocabulary the team uses for lifecycle, programs, and channels. Use these names exactly, so plans, briefs, and the `lifecycle-campaign-strategy` skill all speak the same language. Source: 7shifts Growth Marketing.

## Lifecycle stages

Awareness → Trial & Activation → Onboarding → Expansion → Retention & Re-engagement → Advocacy

## Program types (use these exact labels)

- **Program** — ongoing, always-on motion
- **Campaign** — time-bound, themed push
- **Journey / Flow** — triggered, multi-step automated sequence
- **Experiment** — a test with a hypothesis and a defined read
- **One-Off Send** — a single message, no sequence

Don't mislabel: an Experiment is not a launched initiative, a Campaign is not a Journey. Label fidelity matters (see Source fidelity in [`voice.md`](voice.md)).

## Channels

Email · Modal · In-App Banner · SMS · Push Notification

## Audience state

Every audience has a state: what they know, what they've done, what they're ready for. The strategic test for any campaign is whether its ask matches that state. A TOFU audience that doesn't know the category won't convert to a demo; a fully-adopted customer doesn't need an expansion pitch. (The `lifecycle-campaign-strategy` skill runs on this.)
