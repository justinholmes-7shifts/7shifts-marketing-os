# Clear Communications — Reference

## Rating Scale

| Score | Label | Key Indicator |
|-------|-------|---------------|
| **5** | Exceptional | Main message in first sentence, crystal clear ask |
| **4** | Strong | Main message in first paragraph, minor sharpening needed |
| **3** | Adequate | Main message buried in middle, reader has to work |
| **2** | Needs Work | Builds up to the point, competing messages |
| **1** | Unclear | No identifiable main message or ask |

## Output Format

```
**Rating: [X]/5 - [Label]**

**What's Working:**
- [Specific strength]
- [Specific strength]

**To Reach [X+1]/5:**
- [Specific, actionable change]
- [Specific, actionable change]

**Suggested Opening (Revised):**
[Show the first 2-3 sentences rewritten to lead with the answer]
```

## Quick Reference Patterns

| Type | Structure |
|------|-----------|
| **Recommendation** | We should do X. Reason 1. Reason 2. Here's the plan. |
| **Status Update** | We are [on/behind/ahead]. Progress. Risks. Decisions needed. |
| **Problem + Solution** | We have problem X. Root cause is Y. Fix by doing Z. |
| **Insight** | Key finding is X. Matters because Y. Evidence. Action. |

## Format-Specific Tips

| Format | Lead With |
|--------|-----------|
| **Slack/Email** | First sentence = main message. Bold it. |
| **Slides** | Title = assertion, not topic. Body = evidence. |
| **Loom/Video** | First 15 seconds = topic + key takeaway. |
| **Document** | Executive summary first. Details after. |
