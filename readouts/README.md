# Readouts

Shareable, team-facing outputs. The polished artifacts other people read to understand what's happening: campaign recaps, monthly readouts, planning summaries.

**These pull from the brain.** A readout uses `../context/brand-guidelines.md` for styling and `../context/voice.md` for tone, and cites real results from `../campaigns/` and `../learnings.md`. If you're writing a readout by hand from scratch, the brain isn't doing its job yet.
